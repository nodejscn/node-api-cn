* {number} 超时时间（以毫秒为单位）。**默认值:** `0`（禁用）。

认定套接字超时的不活动毫秒数。

值为 `0` 将禁用传入连接的超时行为。

套接字超时逻辑在连接时设置，因此更改此值仅影响到服务器的新连接，而不影响任何现有连接。
